### 斐波那契数列

**定义：**

**$F_0$ = 0**

**$F_1$ = 1**

**$F_n$ = $F_{n-1}$  +  $F_{n-2}$ ($n≥2$)**

输入整数 n，求斐波那契数列的第 n 项

```cpp
// 算法 1. 递归		时间复杂度：O(2^N)
int Fibonacci(int n) {
    if(n == 0)  return 0;
    else if(n == 1) return 1;
    else
        return Fibonacci(n - 1) + Fibonacci(n - 2);
}

// 算法 2. 
```
